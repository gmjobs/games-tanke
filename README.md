# 坦克大战 · 豪华版（Tank Battle Deluxe）

纯前端、离线可玩的经典 Battle City 网页复刻 —— Canvas 2D 网格射击。单人 / 双人本地合作、10 关推进、6 种道具、计分与最高分、暂停、触屏控件、WebAudio 音效与 localStorage 存档。**无后端、零构建即可运行。**

## 运行方式

| 方式 | 命令 / 操作 |
|------|------|
| 直接打开 | 双击 `index.html`（`file://` 即可游玩） |
| 本地服务器 | `npm run dev` → 打开 http://localhost:5173 |
| 单文件交付 | `npm run build` → 生成自包含 `dist/index.html` |
| 单元测试 | `npm install && npm test`（逻辑层 Vitest 覆盖） |

> `npm install` 仅安装开发期测试依赖（vitest）；游戏本身运行时零依赖。

## 操作

- **玩家 1**：方向键 `↑↓←→` 移动，`空格` 射击
- **玩家 2**（双人）：`WASD` 移动，`F` 射击
- **暂停**：`Esc` 或 `P`；切到后台自动暂停
- **菜单**：`↑↓` 选择、`回车` 确认，或直接鼠标点击
- **触屏**：设置中开启「触屏方向键」，使用屏幕 `▲◀▶▼` 与「射击」按钮

## 道具（击毁闪烁敌人掉落）

| 火 | 盾 | 炸 | 命 | 停 | 固 |
|----|----|----|----|----|----|
| 火力增强(破钢墙) | 限时护盾无敌 | 清屏炸弹 | 加命 +1 | 敌人限时定身 | 基地限时加固 |

## 工程结构

```
index.html        页面骨架 + 内联 CSS（DOM Overlay 菜单/结算）
src/logic.js      逻辑层：地图/碰撞/配置等纯函数（单一事实来源，可单测）
src/game.js       引擎层：固定时间步长主循环 + FSM + 渲染 + 输入 + 音频 + 持久化
src/logic.test.js 逻辑层单元测试（Vitest）
serve.js          零依赖静态服务器（npm run dev）
build.js          单文件内联构建（npm run build → dist/index.html）
```

架构遵循技术方案：**分层解耦**（逻辑/渲染/输入/音频/持久化）、**数据驱动**（关卡/敌人/道具皆为配置）、**固定时间步长**（帧率无关）、**DOM + Canvas 混合**、**localStorage 单键存档**。
